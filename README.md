🚀 RecruitAI – AI Recruitment Assistant

---

📌 Chosen Vertical

AI-Powered Recruiting & Talent Matching

RecruitAI focuses on improving the hiring pipeline by combining AI-driven outreach, intelligent resume screening, and interview automation into a single system.

---

🧠 Approach and Logic

The system is designed to solve three major problems in recruitment:

1. Generic outreach messages → Low response rates
2. Manual resume screening → Time-consuming and biased
3. Lack of scalable interviews → Limited evaluation

Solution Approach:

- Use AI (Google Gemini) to understand candidates and generate personalized communication
- Automate email sending using Google Apps Script + Gmail
- Rank candidates using intelligent evaluation instead of keyword matching
- Enable candidates to interact with AI through interview links

The logic ensures the entire pipeline becomes:
👉 Intelligent → Personalized → Automated → Scalable

---

⚙️ How the Solution Works

🔹 1. Outreach Module

- HR enters candidate details:
  
  - Name, Email
  - Role, Experience
  - Skills
  - LinkedIn (optional)

- AI generates:
  
  - Candidate insights
  - Personalized outreach message
  - Follow-up message

- HR reviews and clicks Send Email

- Email is sent via Gmail using Google Apps Script

---

🔹 2. AI Email Interaction

- Candidate receives a personalized email
- Email includes an AI Interview Link

---

🔹 3. AI Interview Module

- Candidate clicks link

- AI starts interactive interview:
  
  - Asks role-based questions
  - Adapts based on responses
  - Simulates real interview experience

- AI provides:
  
  - Feedback
  - Basic evaluation

---

🔹 4. ATS Scanner Module (Advanced AI Evaluation)

- HR uploads multiple resumes (PDFs)

- Provides job description / required skills

- AI performs deep candidate evaluation using:
  
  - 📌 Project Analysis
    Evaluates project complexity, technologies used, and real-world relevance
  
  - 📈 Growth Analysis
    Understands career progression and learning curve over time
  
  - 🧠 Skill Depth (Not Keywords)
    Identifies how skills are actually applied in projects rather than just listed
  
  - ⚙️ Problem-Solving Indicators
    Infers capability from experience and project outcomes

- ❗ Unlike traditional ATS systems that rely on keyword matching, RecruitAI focuses on quality, capability, and growth potential

- Output:
  
  - Ranked candidate list
  - Best-fit profiles based on real skills and potential

---

🔹 5. Dashboard Tracking

- Tracks candidate status:
  - Sent
  - Replied
  - No Response

---

⚠️ Assumptions Made

- Candidate data is manually entered (no live LinkedIn scraping)
- Email tracking is semi-automatic (manual status update)
- AI interview is a basic conversational prototype
- Resume parsing is simplified for demonstration
- System is designed as a hackathon prototype, not full production

---

🛠️ Tech Stack

- Frontend: React
- AI: Google Gemini API
- Email Automation: Google Apps Script (Gmail)
- Database: Firebase Firestore

---

🎯 Key Highlights

- End-to-end recruitment system
- AI-powered outreach + ATS + interview
- Intelligent evaluation beyond keyword matching
- Fully integrated Google ecosystem
- Real-world use case with automation

---

🚀 Summary

RecruitAI transforms recruitment by:

- Personalizing candidate communication
- Automating outreach and follow-ups
- Ranking candidates using real-world capability analysis
- Enabling AI-driven interviews

It demonstrates how AI can make hiring faster, smarter, and more human-like.
